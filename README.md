### What is this mod?
This is Termites! It's based off Blockixel Artistry's concept for what termites could be.
### What does this mod add?
As you might be able guess, it adds Termites! But that's not all it adds.
#### This mod adds:
- Termites
- Woven Termites
- Gnawed Wooden Sword
- Wooden Helmet
- Sunken Boat Post
- Termite's Nest
- Gnawed Acacia Wood Set
